Content Change Detector

Current selector:  

How to use:

  1. Permission to display notification in the site must be allowed.
  2. Use prefix '.' for class selectors.
     For example: To select <div class="targetname"></div>
                  Use .targetname    
  3. Use prefix '#' for id selectors.
     For example: To select <div id="targetname"></div>
                  Use #targetname

How to stop monitoring:

  Reload the page to remove the content change monitoring.

Warning:

  Do not initialize monitoring more than once. 
  Incorrect usage may cause the browser to crash.
